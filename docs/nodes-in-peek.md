# Nodes in Peek

Individual nodes within the Peek sidebar have their own context menu for quick organization.

![Peek Node Parts](assets/images/6_Peek_Node_Parts.png)

## Node Operations

Clicking the dropdown arrow on a node provides the following options:

![Node Context Menu](assets/images/6_Peek_Node_Context_Menu.png)

1.  **Up/Down Arrows**: Move the node above or below other nodes in the list.
2.  **Remove from Collection**: Stops displaying the node in this collection (does not delete the node itself).
3.  **Rename Node**: Change the display label of the node in the sidebar.
4.  **Move Collection**: Move the node to a different collection or create a new one.
