# Welcome to Peek a Node

**Peek a Node** is the ultimate add-on for Blender that simplifies your node workflow.

## Getting Started

Check out the [Quick Start Guide](quick-start.md) to get up and running in minutes.

![Peek a Node Logo](assets/logo/Peek_a_Node_Logo.png)

## Features

*   **Quick Preview**: valid for Shader, Geometry, and Compositor nodes.
*   **Performance**: Optimized for speed and minimal overhead.
*   **Customizable**: Adjust settings to fit your specific workflow needs.
